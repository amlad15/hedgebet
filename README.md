# HedgeBet IQ

Quant betting assistant built with Streamlit. Implements three finance inspired strategies repurposed for sports betting:
- Statistical Arbitrage (stat arb)
- Volatility Trading (totals)
- Market Making / Middle Scalp

## Run locally
1. Install deps: `pip install -r requirements.txt`
2. Run: `streamlit run hedgebet_iq_app.py`

## Deploy
Recommended: Streamlit Cloud (https://share.streamlit.io)

## Disclaimer
Educational and prototyping tool only. Backtest and use responsibly.
